DoIt is a personal project I developed from scratch using Vue.js 3, showcasing an intuitive and responsive task management application. The app is designed to be sleek, efficient, and user-friendly, focusing on usability and aesthetics.

Project Highlights:
  • Custom-Built Design: A completely original design coded from the ground up.
  • Vue.js State Management: Real-time task updates and seamless user interaction.
  • Accessibility: Responsive layout with the help of Bootstrap 5 and a dark/light mode toggle.

This project reflects my dedication to clean design and functional development in a limited timeframe(3 days!).

Logo Design

Dive into my personal logo—proof that even as a Figma newbie, creativity can shine through! I designed this logo from scratch, blending my front-end chops with a dash of design flair.

Project Highlights:
  • From Scratch Magic: Built this logo from the ground up, showing that you don’t need to be a pro to create something unique.
  • Color Palette: Purple, white, black, and grey were chosen with intention. Purple brings creativity, sophistication, and a touch of mystery, while white provides a modern contrast, and black and grey add a sleek, professional edge.
  • Typography: Lucida Console adds a stylish, monospaced touch that nods to classic terminal fonts, making the design both readable and distinctive.
  • Logo Cover: Crafted to make a memorable impact and reflect my creative approach, ensuring it stands out in any setting

This project demonstrates how fresh ideas and creativity can make a novice designer shine, turning simple concepts into a standout visual identity.
